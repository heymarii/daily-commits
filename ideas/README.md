# Ideas Backlog

When you sit down to commit and your mind is blank — look here.

## TIL Ideas (Always Have Material)

- A CLI tool you used today — what flag did you discover?
- A Python function you looked up — how does it actually work?
- A debugging session — what was the root cause?
- A concept from an article you read — summarize it in your own words
- An AI prompt that worked surprisingly well — document it

## "If All Else Fails" Commits

These are always valid commits when you're in a slump:
1. Fix a typo in any README
2. Add one link to `second-brain/resources/`
3. Add one item to this ideas list
4. Update a learning roadmap checkbox
5. Write a one-paragraph reflection on something you noticed this week
